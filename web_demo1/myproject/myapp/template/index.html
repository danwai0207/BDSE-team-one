Introduction

Problem Definition

via Alibaba Cloud Tianchi competitions

It is important for merchants to identify who can be converted into repeated buyers. By targeting on these potential loyal customers, merchants can greatly reduce the promotion cost and enhance the return on investment (ROI). It is well known that in the field of online advertising, customer targeting is extremely challenging, especially for fresh buyers. However, with the long-term user behavior log accumulated by Tmall.com, we may be able to solve this problem.

A set of merchants and their corresponding new buyers acquired during the promotion on the "Double 11" day.

Purpose: Our task is to predict which new buyers for given merchants will become loyal customers in the future. Here is the data field:
  • User id
  • Age range
  • Gender
  • Merchant id
  • Activity log

Methods

Data Sources

The dataset comes from the Alibaba Cloud Tianchi competitions, and it is related to user behavior log accumulated by Tmall.com. The analysis goal is to predict the probability that these new buyers would purchase items from the same merchants again within 6 months.
